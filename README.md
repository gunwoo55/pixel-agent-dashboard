# Pixel Agent Dashboard

Pixel Lab 에이전트를 고용하고 관리하는 대시보드 시스템

## 🚀 기술 스택

- Next.js 16
- TypeScript
- Tailwind CSS
- Zustand (상태 관리)
- Recharts (차트)

## 📁 프로젝트 구조

```
pixel-agent-dashboard/
├── src/
│   ├── app/              # Next.js App Router
│   ├── components/       # React 컴포넌트
│   ├── store/           # Zustand 스토어
│   └── types/           # TypeScript 타입
├── public/              # 정적 파일
└── .github/workflows/   # GitHub Actions
```

## 🎮 기능

- **대시보드**: 실시간 통계, 에이전트 현황
- **에이전트 관리**: 고용/해고, 스킬 확인
- **에이전트 마켓**: 새 에이전트 고용
- **작업 보드**: 칸반 스타일 작업 관리
